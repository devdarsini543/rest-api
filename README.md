# 🧩 Simple REST API with Flask

This is a beginner-friendly REST API built using **Python** and **Flask** to manage user data with basic `GET`, `POST`, `PUT`, and `DELETE` functionality.

---

## 🚀 Features

- View all users (`GET`)
- Add new users (`POST`)
- Update user data (`PUT`)
- Delete user (`DELETE`)

---

## 🔧 Tech Stack

- Python 🐍
- Flask 🌶️
- Postman / Curl 🌐
